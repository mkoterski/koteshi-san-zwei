Check CPU temperature via terminal
Source: https://phoenixnap.com/kb/linux-cpu-temp
- install via: sudo apt install lm-sensors
- To check SSD and hard drive temperatures, execute the following command: sudo hddtemp /dev/sda
- To see which system components you can monitor, run: sudo sensors-detect
- To run the sensors command repeatedly and get real-time data in the terminal, execute the following command: watch sensors

Check GPIO status
- The "raspi-gpio get" command will list the status of all GPIOs (not just those available on the connector).
- Limit it with "raspi-gpio get 2-27" to just list the ones available for general use.

Command: echo "pud 4 d" >/dev/pigpio
Return: GPIO 4: level=0 fsel=0 func=INPUT pull=DOWN
